Sample Portal Landing Page

This is a sample landing page for the portal.

<button onclick="rh24RedirectMessage('/login')">
    login
</button>
<button onclick="rh24RedirectMessage('/signup')">
    register
</button>
<script>
    function rh24RedirectMessage(path) {
        window.parent?.postMessage({
            type: 'RH24_Change_Location',
            relativeUrl: path
        }, '{rhodium24-base-url}')
    }
</script>
