# RH24-WebApp-SDK React Sample

create-react-app bootstraped sample of using [**rh24-webapp-sdk**]() to run [Rhodium24](rhodium24.io) in an existing website.

## instalation
``` bash
npm install
or
yarn install
```

## build
``` bash
npm build
or
yarn build
```

## run
``` bash
npm start
or
yarn start
